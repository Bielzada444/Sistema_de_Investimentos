Descrição detalhada

Cabeçalho unificado para inclusão de todas as classes de domínio do sistema.

Agrupa todas as classes de domínio necessárias para o sistema, garantindo acesso centralizado às regras de validação
e estruturas de dados específicas. Ideal para simplificar includes em outros módulos do projeto.

Domínios Incluídos:

  • Codigo: Identificador numérico único (5 dígitos, ex: 12345)
  • CodigoNegociacao: Código de negociação padrão B3 (4 letras + 1 número, ex: PETR4)
  • Cpf: Validador de CPF (formato XXXXXXXXXXX)
  • Data: Manipulação de datas (formato AAAAMMDD)
  • Dinheiro: Valores monetários (ex: R$ 1,234.56)
  • Nome: Nomes de usuário/entidades (1-20 caracteres alfabéticos)
  • Perfil: Perfis de risco (Agressivo, Moderado, Conservador)
  • Quantidade: Quantidades de ativos (100-50000 unidades)
  • Senha: Senhas seguras (6 caracteres com mix case e símbolos)
    Observação
    Inclui DominioException para tratamento de erros específicos (exceções lançadas durante validações).
