Descrição detalhada

Classe responsável por armazenar e validar um código numérico de exatamente 5 dígitos (0-9).

Esta classe verifica se o código inserido segue o formato correto, garantindo que ele contenha apenas dígitos numéricos e tenha o tamanho esperado. Caso contrário, impede seu armazenamento.
