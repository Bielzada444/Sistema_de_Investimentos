Descrição detalhada

Classe responsável por armazenar e validar um código de negociação.

Esta classe permite o armazenamento de um código alfanumérico de até 12 caracteres, garantindo que ele siga as regras de formatação estabelecidas. O código pode conter letras maiúsculas (A-Z), letras minúsculas (a-z), números (0-9) e espaços.
