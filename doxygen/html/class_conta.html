Descrição detalhada

Esta classe representa o registro do usuário no sistema de investimentos.

Agrega os dados essenciais para autenticação:

  • CPF (identificador único do titular)
  • Nome do usuário (para exibição e verificação)
  • Senha (credencial de acesso com requisitos de segurança)
