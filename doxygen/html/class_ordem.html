Descrição detalhada

Esta classe representa a ordem de compra/venda de ativos no sistema de investimentos.

Garante integridade dos dados através de domínios especializados que validam:

  • Formato de códigos
  • Datas cronológicas
  • Valores monetários
  • Quantidades negociáveis
    Integrado com exceções específicas (DominioException) para tratamento de erros.

◆ setCodigoNegociacao()

Como registrar o ativo financeiro:

  • Use o código do ativo (ex: "VALE3" para ações da Vale)
  • Pode conter até 12 letras/números (ex: "OUROPRT", "DOLAR")

◆ setData()

void Ordem::setData ( const Data & novaData)

Como definir a data da operação:

  • Formato: AAAAMMDD (ex: "20240507" para 07/05/2024)
  • O sistema bloqueia datas inválidas ou futuras

◆ setDinheiro()

void Ordem::setDinheiro ( const Dinheiro & novoPreco)

Como informar o preço:

  • Valor unitário (ex: R$ 72.50 por ação)
  • Aceita de R$ 0.01 até R$ 1,000,000.00
  • O sistema ajusta para 2 casas decimais automaticamente

◆ setQuantidade()
