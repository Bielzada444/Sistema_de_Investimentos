Descrição detalhada

Classe de teste de unidade para o domínio Dinheiro.

Verifica se o domínio Dinheiro aceita corretamente valores válidos, realiza arredondamento automático e rejeita valores inválidos.

Documentação das funções

◆ runDinheiro()

int TUDinheiro::runDinheiro ( )

Executa todos os testes do domínio Dinheiro.

Verifica:

  • Cenário de sucesso com um valor monetário válido (R$ 0,01 a R$ 1.000.000,00).
  • Cenário de falha com valores negativos, zero ou acima do limite.
  • Arredondamento correto para 2 casas decimais.
    Retorna
    SUCESSO se todos os testes passarem, FALHA caso contrário. Direciona a execução de todos os testes
