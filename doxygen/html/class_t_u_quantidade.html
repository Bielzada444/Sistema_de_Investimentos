Descrição detalhada

Classe de teste de unidade para o domínio Quantidade.

Valida:

  • Quantidades inteiras entre 1 e 1,000,000 unidades;
  • Rejeição de valores zero, negativos ou acima do limite;
  • Conversão correta de strings para valores numéricos.
